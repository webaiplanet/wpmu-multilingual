# WPMU Multilingual 文档目录

此目录按记录类型分文件夹存放，后续新增文档优先放到对应目录，不要全部堆在 `docs/` 根目录。

## 目录规则

- `reference/`：长期有效的架构、规则、设计说明。
- `changelog/`：版本更新日志、变更记录。
- `plans/`：开发方案、需求说明、待实现功能说明。
- `reports/`：测试报告、审核报告、问题处理报告、阶段汇报。
- `logs/`：过程日志、排查记录、持续改进记录。

## 当前重点文档

- Term 增删改同步需求：`plans/TERM_SYNC_REQUIREMENTS.md`
- 文章关联改进方案：`plans/WPMU多语言插件_文章关联改进方案.md`
- 架构与翻译规则：`reference/ARCHITECTURE_AND_TRANSLATION_RULES.md`
- 更新日志：`changelog/CHANGELOG.md`
