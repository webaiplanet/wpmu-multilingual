# WPMU 多语言插件 0.9.8.12 后台 UI 文案清理报告

## 处理目标

清理后台公开界面中遗留的开发阶段说明，并统一 OpenAI 兼容页二级选项卡样式。

## 已处理内容

- OpenAI 兼容页二级选项卡改为与“语言切换”页一致的标签式 UI。
- 删除或改写以下类型文案：
  - “现在使用自己的二级选项卡”
  - “从翻译设置独立出来”
  - “后续继续增加”
  - 带具体历史版本号的开发说明
  - 面向开发维护的 README 同步要求
- 模型示例改为通用 OpenAI 兼容示例。
- 语言切换、翻译引擎、OpenAI、人工翻译、帮助页说明统一改为面向插件管理员的配置说明。
- 工具页移除已停用的同 ID 猜测补建按钮；直接请求旧动作会明确返回“工具已停用”。

## 验收建议

- 打开“翻译引擎 → OpenAI 兼容”，确认二级选项卡样式与“语言切换”的二级选项卡一致。
- 检查翻译引擎页顶部说明，不再出现开发迁移说明。
- 检查 OpenAI 页说明，不再出现“自己的二级选项卡”等开发文案。
- 检查帮助页，不再出现源码维护要求。
- 检查工具页，不再出现同 ID 猜测补建按钮。
