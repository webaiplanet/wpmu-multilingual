# WPMU 多语言插件 0.9.8.13 语言切换代码示例样式修复报告

## 问题

“语言切换 → 代码调用”中的 PHP 示例代码显示成一行，并把 `\n` 当普通文本显示，阅读和复制都不方便。

## 原因

示例代码使用单引号 PHP 字符串输出，字符串里的 `\n` 没有被转换为真实换行。

## 处理结果

- 示例代码改为 PHP 多行字符串变量。
- 输出改为 `<pre><code>`。
- 统一代码块样式：
  - 等宽字体
  - 合理行高
  - 边框和内边距
  - 保留换行与缩进
  - 内容过宽时横向滚动

## 验收建议

- 打开“语言切换 → 代码调用”。
- 确认 PHP 示例按多行显示。
- 确认代码缩进正常，不再显示字面量 `\n`。
- 确认长代码可横向滚动。
